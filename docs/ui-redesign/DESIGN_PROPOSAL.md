# SatoTrip UI リニューアル提案 v2 — 「趣味 × 地元インサイダー」

**種別**: デザイン方向性の提案（コードの本適用は含みません）
**成果物**: 本ドキュメント ＋ 単体で開けるモックアップ `docs/ui-redesign/mockup.html`

> v1（地域固有の「みちしるべ／火山と湾」）から、プロダクトビジョンに合わせて刷新した版です。
> ビジョン: **全国展開 / 趣味嗜好で検索（ツーリング・カフェ巡り等）/ ニッチなプラン / 地域密着・地元ならではの情報**。

## 0. まず「今やるべきか」

**結論: “デザインを決めて試作する”のは今が適期。ただし全画面の本適用は段階的に。**

機能面はこの数日で大きく安定しました（重大バグ修正・新機能・E2E検証・耐障害性強化）。土台が固まった今は見た目への投資が妥当です。一方、全画面の一括差し替えは早い。先に **前提作業**（Tailwind/ReactのCDN依存 → Viteビルド一本化＋トークン定義）を済ませ、**コア画面から段階移行**するのが安全です。本提案は `main` と別ブランチに隔離しており、進行中の改修に影響しません。

## 1. コンセプト: 「趣味の解像度で、日本を歩く。」

SatoTrip の差別化は **「定番」ではなく「あなたの好き」で旅を組む**こと。UIの主役は美しい写真カードではなく、**趣味カテゴリ（軸）× 地元インサイダー情報 × 全国どこでも成立する動線**です。

3つの柱を UI に落とし込みます:

1. **趣味で検索（カテゴリカルな軸）** — ツーリング／カフェ巡り／サウナ・温泉／写真映え／歴史／自然／グルメ／アート…。各趣味が **色＋アイコン** を持ち、選ぶと動線・スポット・並びが変わる。“意味＝色”。
2. **地元ならでは（インサイダー）** — 各スポットに **地元の人・常連の“ひとこと”**（明朝で人の声）、看板のない店や知る人ぞ知る道を **「穴場」タグ** で可視化。
3. **全国 × ニッチ** — 特定地域の色に縛らない中立のブランド基調。ニッチな趣味×地域の組み合わせほど強い。

## 2. カラートークン

### ブランド基調（中立・全国で成立）
色はカテゴリに任せ、クロームは静かに保つ（「大胆さは一箇所に集中」）。
| トークン | 用途 | HEX(light) |
|---|---|---|
| `--ground` | 背景（中立の紙） | `#ECEDE8` |
| `--surface` | カード面 | `#FBFBF8` |
| `--ink` | 本文・主要CTA（炭） | `#21262A` |
| `--ink-soft` | 補助テキスト | `#5B6469` |
| `--line` | 罫線 | `#DBDCD5` |
| `--brand` | ブランドマーク・リンク（藍） | `#274690` |

### 趣味カテゴリ・パレット（検索の主役 / カテゴリカル）
各趣味に固有色。ライト/ダーク両対応、コントラスト確保。
| 趣味 | 色(light) | アイコン |
|---|---|---|
| ツーリング | `#4E6E8E` | 🏍️ |
| カフェ巡り | `#B5793C` | ☕ |
| サウナ・温泉 | `#2F8E8E` | ♨️ |
| 写真映え | `#C0567E` | 📷 |
| 歴史・史跡 | `#9A7B3C` | ⛩️ |
| 自然・絶景 | `#5C8A5A` | ⛰️ |
| ご当地グルメ | `#C64B37` | 🍜 |
| アート・クラフト | `#7A5AA6` | 🎨 |

ダークは各色を明度上げした対（例: ツーリング `#6E92B4`）。**セマンティック（good/warn/critical）はカテゴリ色とは別管理**。

## 3. タイポグラフィ（外部CDN非依存・システムフォント）

- **見出し (ゴシック)**: `Hiragino Sans / Yu Gothic` を太ウェイト＋詰めで、現代的で確信のある声。
- **本文 (ゴシック)**: 標準ウェイト、行間 1.7。
- **データ (Mono)**: `ui-monospace / SF Mono` — 時刻・距離(42km)・標高差(+520m)・座標。ツーリングの“ルート感”を担う。
- **地元の声 (明朝)**: `Hiragino Mincho ProN / Yu Mincho` — 常連・地元の人の“ひとこと”だけ明朝にして、人の言葉・編集的な温度を出す（クリーム＋セリフの量産感を避けた、意味に紐づく使い方）。

ゴシック見出し × 等幅データ × 明朝の地元の声、という **役割の対比**が個性です。

---

## 4. レイアウトと主要コンポーネント

- **趣味ファセット選択**（署名的UI）: 色付きチップの列。ヒーローと検索の中心に置く。
- **プランカード＝縦タイムライン**: 時刻(等幅) → スポット → 移動コネクタ。趣味で形が変わる（カフェ＝徒歩ループ「↳徒歩7分・路地裏の階段坂」／ツーリング＝ルート「↳やまなみ 42km・稜線」「↳峠 18km・+520m」）。
- **地元の声ブロック**: 明朝の引用＋出典（「谷中在住・焙煎12年/佐藤さん」）。
- **穴場バッジ**: 看板のない店・地元だけの道を可視化。
- **趣味カテゴリのグリッド**: 各カテゴリに件数（等幅）を添え、規模と網羅性を示す。
- **モーション**: 控えめ・目的的。`prefers-reduced-motion` 尊重。

## 6. 非目標・リスク・次アクション

- 本提案はコードの本適用を含みません（`main`・進行中の改修に非干渉）。
- モックの絵文字・件数はプレースホルダ。実装では登録スポットの実データ／アイコンセットに置換。
- 「地元の声」は**データ収集の方針**（誰が・どう集めるか）とセットで設計が必要（編集・監修フロー）。
- **次アクション候補**: この方向でよければ **P0 基盤**（CDN→ビルド化＋トークン）から着手。色・写真比重・ミニマル度の調整、または3案比較も可能。
